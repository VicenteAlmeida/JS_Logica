Escreva um programa em JS que funcione como um conversor de medidas. O progrtama deverá pedir por um valor em metrso e então dar a opção de escolher para qual unidade de medida esse valor deve ser convertido.A opções são:

  • Milímetro(mm)
  • Centímetro(cm)
  • Decímetro(dm)
  • Decâmetro(dam)
  • Hectômetro
  • Quilômetro

O programa deve então converter a medida de acordo com a opção escolhida e exibir o resultado.

O progrma também deve exibir uma mensage, de "Opção Inválida" caso o usuário insira uma opção diferente das disponíveis (use break e o default para isso)
