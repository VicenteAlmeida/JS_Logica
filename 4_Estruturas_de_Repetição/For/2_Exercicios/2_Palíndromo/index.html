Esvreva um programa em js que seja capaz de identificar se uma palavra é um palíndromo.Um palíndromo é uma palavra que lida de trás para frente possui as mesas letras na mesma ordem.O programa deve iniciar pedindo que seja informada uma palavra e então deve exibir uma mensagem dizendo se aquela palavra é ou não um palíndromo.Caso não seja,o programa deve mostrar a palavra lida da esquerda para direita e da direita para esquerda.
